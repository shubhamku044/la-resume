## Features ✨

- 🎯 ATS-Optimized Templates
- 📤 One-click PDF/LaTeX Export
- 🆓 100% Free & Open Source
- 🔒 No Signups Required
- 🌐 Progressive Web App (PWA) Support

## Quick Start 🚀

### Prerequisites

- Node.js 20+
- pnpm 9+

### Installation

```bash
# Clone repository
git clone https://github.com/shubhamku044/la-resume.git

# Install dependencies
cd la-resume
pnpm install

# Start development server
pnpm dev
```
